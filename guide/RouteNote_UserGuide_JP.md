## はじめに

「軌跡メモ（RouteNote）」は、GPSを活用した軌跡記録・メモ管理アプリです。移動ルートをリアルタイムで記録しながら、地図上の任意の場所にカラーピン付きメモや写真を残すことができます。営業先の管理、旅行の記録、配送ルートの確認など、さまざまな用途でご活用いただけます。

---

## メイン画面

アプリを起動すると、Apple Mapを使ったフルスクリーンの地図画面が表示されます。

### 画面構成

**上部エリア**
- **ユーザー追跡ボタン** -- 現在地を地図の中心に追従させます。タップするたびに「追従なし → 追従 → 方角追従」が切り替わります。
- **コンパスボタン** -- 地図を回転させた際に表示され、タップすると北向きに戻ります。
- **2Dボタン** -- 地図を3D表示（傾き）にした場合に表示され、タップすると平面の2D表示に戻ります。
- **地図タイプボタン** -- 地図の表示形式を切り替えます（標準 / 航空写真 / ハイブリッド）。航空写真とハイブリッドはPlus版の機能です。
- **インフォ（i）ボタン** -- 設定画面を開きます。

**下部ツールバー**
- **ルート一覧** -- 保存済みのルート一覧画面を開きます。
- **記録開始** -- GPSによるルート記録を開始します。
- **終了** -- 記録中のルートを保存して終了します。
- **メモ追加** -- 現在地または任意の位置にメモピンを追加します（記録中のみ使用可）。
- **ルート作成** -- 手動でのルート作成、またはルート結合を行います。

### 記録中の表示

記録中は画面上部に「記録中[タイトル]」と表示されます。タップすると記録中の移動距離を確認できます。

## ルートの記録

### 記録の開始

1. 下部ツールバーの **「記録開始」** ボタンをタップします。
2. ルートのタイトルを入力するダイアログが表示されます。タイトルを入力して「OK」をタップします。
3. GPSによる位置情報の記録が開始されます。設定により、出発地点に自動的にピンが追加されます。
4. 地図上にリアルタイムで軌跡が青いラインとして描画されます。

### 記録中の操作

- 移動距離がリアルタイムで表示されます（メートルまたはマイル）。
- メモピンの追加が可能です（後述）。
- アプリをバックグラウンドにしても記録は継続されます。

### 記録の停止

1. **「終了」** ボタンをタップします。
2. 確認ダイアログが表示されます。「保存」をタップすると、ルートがCore Dataに保存されます。
3. 設定により、到着地点に自動的にピンが追加されます。
4. 保存後、記録したルートを地図上に表示するかどうか確認されます。

### 一時保存

記録停止ボタンを **長押し** すると、記録を終了せずに現在までのデータを一時保存できます。正常に停止した場合、一時保存データは破棄されます。

## メモ（ピン）の追加

記録中に **「メモ追加」** ボタンをタップすると、以下のメニューが表示されます。

### メモの種類

**テキストメモ**
- **「メモ」** -- 現在地にテキスト・写真・色を指定したメモを作成します。
- **「位置指定メモ」** -- 地図上に十字線が表示され、任意の位置にメモを配置できます。地図をスクロールしてピンを置きたい位置を十字線の中心に合わせ、「ここに追加」をタップします。 このメモを使用するには、設定/ルート記録でメモの位置を指定をオンにします。

**カラーピン（クイック追加）**
テキストを入力せずに、現在地にカラーピンをすばやく追加できます。

| 色 | 無料版 | Plus版 |
|---|:---:|:---:|
| 緑 | ○ | ○ |
| 黄 | ○ | ○ |
| 赤 | ○ | ○ |
| 青 | ○ | ○ |
| オレンジ | ○ | ○ |
| 紫 | ○ | ○ |
| シアン | - | ○ |
| 茶 | - | ○ |
| マゼンタ | - | ○ |

各色には設定画面でカスタムラベル名を付けることができます（例：「訪問済み」「要再訪」など）。

## ルート一覧

下部ツールバーの **「ルート一覧」** をタップすると、保存済みのルート一覧が表示されます。

### セクション構成

一覧は3つのセクションに分かれています。

| セクション | 内容               |
|-------|------------------|
| 軌跡    | GPSで記録した実際の移動ルート |
| 計画ルート | 手動で作成したルート       |
| 計画エリア | 手動で作成したエリア（ポリゴン） |

### 表示モード

画面左上のファイルボタンで表示モードを切り替えられます。

- **すべて** -- 全ルートを表示（3セクション）
- **ブックマーク** -- ブックマークしたルートのみ表示
- **履歴** -- 最近表示したルートを表示
- **絞り込み（ルート/エリア）** -- 条件を指定してルートを絞り込み
- **絞り込み （ピン）** -- ピンのテキストや住所で絞り込み（Plus版）

### 並び替え

ファイルボタンから以下の並び替えが可能です。
- 日付降順（新しい順、デフォルト）
- 日付昇順（古い順）
- タイトル降順
- タイトル昇順

### 各ルートの表示内容

- タイトル（未設定の場合は「No title」）
- 日時と出発地の住所
- ブックマークマーク（ブックマーク済みの場合）
- 選択状態のチェックマーク（赤：アクティブ表示、デフォルト色：パッシブ表示）

### スワイプ操作

**左スワイプ（右から左）**
- **削除** -- ルートを完全に削除します（関連する写真・通知も削除されます）。
- **複製** -- ルートのすべてのデータ（メモ・写真含む）を複製します。

**右スワイプ（左から右）**
- **ブックマーク** -- ルートをブックマークに追加/削除します。

### その他の操作

- **読み込み** -- ファイルからルートデータをインポートします。 （Plus版）
- ルートをタップすると、地図上にそのルートのラインとメモピンが表示されます。
- 情報ボタン（i）をタップすると、ルート詳細画面を開きます。

## ルート詳細画面

ルート一覧の情報ボタン（i）またはログテーブルのタイトルをタップすると、ルート詳細画面が表示されます。

### 表示情報

| 項目 | 内容 |
|---|---|
| タイトル | ルートのタイトル（編集可能） |
| 出発時刻 | 記録開始の日時 |
| 到着時刻 | 記録終了の日時 |
| 出発地住所 | 出発地の住所（再読込・編集可能） |
| 到着地住所 | 到着地の住所（再読込・編集可能） |
| 所要時間 | 出発から到着までの経過時間 |
| 距離 | 移動距離（メートルまたはマイル） |
| コメント | ルートに対するコメント |

### 地図表示

- ルートがポリラインとして地図上に表示されます。
- エリアの場合はポリゴン（塗りつぶし）で表示されます。
- 各メモピンがカラーマーカーとして表示されます。

### 操作

- **編集** -- タイトルを変更できます。
- **コメント** -- ルートにコメントを追加・編集できます。
- **メモ並べ替え** -- メモの表示順をドラッグで変更できます。
- **エクスポート** -- RNOTE / JSON / GPX / テキスト形式でルートデータを送信できます。

---

## メモの編集

地図上のメモピンをタップし、吹き出しの情報ボタンをタップするとメモ編集画面が開きます。

### 編集可能な項目

- **テキスト** -- メモの本文を編集できます。
- **色** -- 9色（無料版は6色）から選択できます。
- **住所** -- 自動取得された住所を再読込、またはPlus版で手動編集できます。
- **写真** -- カメラまたはフォトライブラリから写真を添付できます（最大4枚、2枚目以降はPlus版）。

### その他の操作

- **削除** -- メモを削除します。
- **到着通知** -- そのメモの位置に近づいた際に通知を受け取る設定ができます。
- **共有** -- 以下の方法でメモを共有できます。
  - マップアプリで開く
  - マップアプリでナビゲーション（徒歩）
  - テキストと写真を他のアプリに共有

## 写真の添付

各メモには写真を添付できます。

### 写真の追加方法

メモ編集画面で画像エリアをタップすると、以下の選択肢が表示されます。

1. **フォトライブラリ** -- 端末に保存されている写真から選択します。
2. **カメラ** -- その場で写真を撮影します（Plus版ではピンチでズーム対応）。

### 写真の管理

- 最大 **4枚** までの写真を添付できます（2枚目以降はPlus版の機能）。
- 写真はタップすると全画面表示になり、ピンチでズームできます。
- 全画面表示から **共有** や **削除** が可能です。
- 設定で写真サイズを「640x480」に制限でき、ストレージ容量を節約できます。
- 設定で「フォトライブラリに保存」をオンにすると、撮影した写真が端末のカメラロールにも保存されます。

## ルート作成（手動）

GPSで記録する以外に、地図上でタップしてルートを手動で作成することができます。

### 作成手順

1. 下部ツールバーの **「ルート作成」→「ルート作成」** をタップします。
2. ルート作成画面が開きます。
3. **「ルート」** をタップします。
4. 地図をスクロールして次の地点を画面中央に合わせます。
5. 以下のいずれかのボタンで地点を追加します。
   - **「ルートを追加」** -- 前の地点から実際の道路に沿ったルートを検索して描画します。
   - **「直線を追加」** -- 前の地点から直線を引きます。

### 移動手段の選択

上部のセグメントコントロールで移動手段を切り替えられます。
- **徒歩** -- 歩行ルートを検索
- **車** -- 自動車ルートを検索

### メモの追加

ルート作成中でもメモピンを追加できます（緑・黄・赤の3色）。

### 保存

「保存」をタップしてタイトルを入力すると、ルートが「ルートプラン」として保存されます。

### 既存ルートの参照

ルート作成中に既存のルートを参考として地図上に表示（パッシブ表示）することができます。

---

## エリア作成

地図上にポリゴン（エリア）を作成して、特定の区域を管理できます。

### 作成手順

1. ルート作成画面で **「エリア」** をタップします。
2. 3点以上の地点を「ルートを追加」または「直線を追加」で配置します。
3. **「エリアを完了」** をタップすると、最後の地点と最初の地点が接続され、ポリゴンが完成します。
4. タイトルを入力して保存します。

エリアは「エリアプラン」として保存され、統計機能でピンの集計に利用できます。

---

## 行政区域の取得（市区町村・都道府県など）

OpenStreetMapのデータを活用して、行政区域の境界線を自動的に取得してエリアを作成できます。

### 使用方法

1. ルート作成画面で **「地域」** をタップします。
2. 取得したい地域の付近に地図の中心を合わせます。
3. 以下の6段階の区分レベルから選択します。

| レベル | 範囲 | Plus版要否 |
|---|---|:---:|
| 地域 | 広域の地域区分 | 無料 |
| 都道府県 | 都道府県の境界 | Plus版 |
| 郡 | 郡の境界 | Plus版 |
| 市区町村 | 市区町村の境界 | Plus版 |
| 区 | 区・地区の境界 | Plus版 |
| 町内 | 町丁目レベルの境界 | Plus版 |

4. 複数のポリゴンが返される場合（離島がある場合など）は、前へ/次へボタンで切り替えられます。
5. 「保存」をタップすると表示中のエリアが保存されます。

> **注記**: 検索データは (c) OpenStreetMap contributors によるものです。

## ルート結合

複数のルートを1つのルートにまとめることができます（Plus版の機能）。

### 結合手順

1. 下部ツールバーの **「ルート作成」→「ルート結合」** をタップします。
2. 結合対象のルートを2つ以上選択します。
3. 選択したルートが色分けされて地図上にプレビュー表示されます。
4. **「結合」** をタップすると、すべてのルートの軌跡データとメモが1つにまとめられます。
5. タイトルを入力して保存します。

結合されたルートには、元のルートのすべてのメモと写真が引き継がれます。

## 検索とフィルター

### ルート検索

ルート一覧画面で「検索」モードを選択すると、以下の条件でルートを絞り込めます。

- **タイトル** -- ルートのタイトルに含まれるテキスト
- **コメント** -- ルートのコメントに含まれるテキスト
- **出発地住所** -- 出発地の住所に含まれるテキスト
- **到着地住所** -- 到着地の住所に含まれるテキスト

複数の検索条件を保存でき（最大5件）、条件はOR（いずれかに一致）で結合されます。各条件内の複数フィールドはAND（すべてに一致）で評価されます。

### ピン検索（Plus版）

「ピン検索」モードでは、メモ（ピン）の内容でルートを絞り込めます。

- **ピンテキスト** -- メモのテキストに含まれる文字列
- **ピン住所** -- メモの住所に含まれる文字列

## 色フィルター

地図上のメモ一覧パネルでは、色ごとにメモの表示/非表示を切り替えられます。

### 使用方法

1. 地図上のメモ一覧パネルのメニューボタンをタップします。
2. **「ピン色で絞り込み」** を選択します。
3. 各色のチェックをオン/オフして、表示するメモを選択します。
4. **「すべて表示」/「すべて非表示」** で一括切り替えが可能です。

フィルターが有効な場合、ログタイトルにアスタリスク（*）が表示されます。

### テキストフィルター（Plus版）

メモ一覧パネルのメニューから **「テキストで絞り込み」** を選択すると、メモのテキスト内容でフィルタリングできます。

## 到着通知

特定のメモの位置に近づいた際に通知を受け取ることができます。

### 設定方法

1. メモ編集画面で **ベルアイコン** をタップします。
2. 通知が有効になると、そのメモの位置を中心に半径100mの監視領域が設定されます。
3. 端末がその領域に入ると、カスタム通知音とともに通知が届きます。

### 制限事項

- 同時に監視できる地点は最大 **20箇所** です。
- 設定 → 「到着通知一覧」で、現在登録されている通知を確認・管理できます。

## データ転送（エクスポート）

ルートデータを4つの形式でエクスポートできます。

### エクスポート形式

| 形式 | 拡張子 | 内容 | Plus版要否 |
|---|---|---|:---:|
| RNOTE | `.rnote` | 圧縮形式（JSON + 写真をZIP圧縮） | Plus版 |
| JSON | `.json` | ルート・メモ・写真データをJSON形式で保存 | 無料 |
| GPX | `.gpx` | GPS交換フォーマット（他のアプリとの互換性あり） | 無料 |
| テキスト | `.txt` | ルート情報とメモをプレーンテキストで出力 | 無料 |

### エクスポート手順

1. **設定 → データ転送** から形式を選択します。
2. エクスポートするルートをタップします。
3. JSON形式で写真が含まれる場合は、画質（高・中・低）を選択できます。
4. 共有シートが表示され、AirDrop・メール・メッセージなどで送信できます。

ルート詳細画面からも個別のルートをエクスポートできます。

---

## データの読み込み（インポート）

他のデバイスや他のアプリからルートデータを読み込むことができます。

### 対応形式

| 形式 | 拡張子 | 説明 |
|---|---|---|
| JSON | `.json` | 軌跡メモのJSON形式 |
| RNOTE | `.rnote` | 軌跡メモの圧縮形式 |
| GPX | `.gpx` | 汎用GPS交換フォーマット |

### インポート方法

1. **ファイルアプリからの読み込み** -- 他のアプリからファイルを「軌跡メモ」で開きます。
2. **ルート一覧 → 読み込み** -- デバイス内のファイルを選択して読み込みます。
3. **AirDrop** -- 他のデバイスからAirDropで受け取ります。

インポートされたルートは「Copy of: [元のタイトル]」または「[ファイル名]からのインポート」というタイトルで保存されます。

GPXファイルは、軌跡メモ以外のアプリで作成されたものでも読み込むことができます。

## 統計機能

### エリア統計

選択したエリア内に含まれるピンの数を色別に集計します。

1. **設定 → エリア統計** を開きます。
2. 集計対象のエリアにチェックを入れます。
3. 各色ごとに「エリア内のピン数 / 全ピン数」が表示されます。

配送計画や営業戦略の分析に活用できます。

### ルート統計（Plus版）

選択したルートに含まれるピンの数を色別に集計します。

1. **設定 → ルート統計** を開きます。
2. 集計対象のルートにチェックを入れます。
3. 各色ごとのピン数が表示されます。

## 設定

メイン画面左上の **インフォ（i）ボタン** から設定画面を開けます。

### ルート記録

| 設定項目 | 説明 |
|---|---|
| カラーラベル設定 | 各色ピンのデフォルトラベル名を設定 |
| 出発ピンの追加 | 記録開始時に出発ピンを自動追加（オン/オフ） |
| 到着ピンの追加 | 記録終了時に到着ピンを自動追加（オン/オフ） |
| メモ追加位置の指定 | 十字線で任意の位置にメモを追加可能にする（Plus版） |
| 位置フィルター | 位置情報の精度フィルタリング（オン/オフ） |
| 位置検出間隔 | GPSサンプリング間隔（最短〜5秒） |

### 写真

| 設定項目 | 説明 |
|---|---|

## トラブルシューティング

### 位置情報が取得できない

- 設定アプリ → プライバシーとセキュリティ → 位置情報サービスで「軌跡メモ」が「常に許可」になっていることを確認してください。
- バックグラウンドでの記録には「常に許可」が必要です。

### 記録が途切れる

- 位置検出間隔を「最短」に設定してみてください。
- 「位置フィルター」をオフにしてみてください。

### データの送受信が表示されない

- データが大きい場合(特に写真が多い場合)、転送が途中で止まることがあります。
- 圧縮転送またはGPXで転送をお試しください（GPX転送では写真を別に転送する必要があります）。
